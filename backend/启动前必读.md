## 🔧 首次启动步骤

### 第一步：准备环境

```bash
# 1. 确保在backend目录
cd backend

# 2. 确保Docker服务运行
docker-compose up -d
docker-compose ps

# 3. 配置.env文件
cp env.example .env
# 编辑.env，填入ZAI_API_KEY
```

### 第二步：初始化数据库

**重要**：必须先初始化数据库！

```bash
poetry run python reset_db.py
```

或者如果uvicorn没有运行：

```bash
poetry run python init_db.py
```

### 第三步：启动服务

```bash
poetry run uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 第四步：验证

访问: http://localhost:8000/docs

或测试健康检查:

```bash
curl http://localhost:8000/api/v1/system/health
```

## ⚠️ 常见错误

### ❌ `no such column: novels.processing_progress`

**原因**: 数据库表结构不完整

**解决**:
1. 停止uvicorn (Ctrl+C)
2. 运行 `poetry run python reset_db.py`
3. 重新启动uvicorn

### ❌ `error parsing value for field "CORS_ORIGINS"`

**原因**: .env文件中CORS_ORIGINS格式不对

**正确格式** (不要用引号):
```bash
CORS_ORIGINS=http://localhost:5173,http://127.0.0.1:5173
```

**错误格式**:
```bash
CORS_ORIGINS="http://localhost:5173, http://127.0.0.1:5173"  # ❌
```

### ❌ `SQLite database is locked`

**原因**: 有其他uvicorn进程在使用数据库

**解决**:
1. 找到所有Python进程: `tasklist | findstr python`
2. 结束进程: `taskkill /F /PID <进程ID>`
3. 或重启电脑
