## 后台任务问题排查

### FastAPI BackgroundTasks 的限制

FastAPI 的 `BackgroundTasks` 有以下限制：
1. **不可靠**：如果服务器重启，任务会丢失
2. **无重试机制**：失败后不会自动重试
3. **无监控**：难以跟踪任务状态
4. **内存限制**：在 FastAPI 进程中运行，共享内存

### 建议改进

1. **使用 Celery**：更可靠的异步任务队列
2. **添加重试逻辑**：自动重试失败的任务
3. **改进错误处理**：捕获并记录详细的错误信息
4. **添加任务监控**：实时监控处理进度

## 临时解决方案

### 创建重新处理端点

添加一个 API 端点来重新处理失败的小说：

```python
@router.post("/{novel_id}/reprocess", response_model=NovelProcessingResponse)
async def reprocess_novel(
    novel_id: str,
    background_tasks: BackgroundTasks,
    novel_service: NovelService = Depends(get_novel_service),
    text_service: TextProcessingService = Depends(get_text_processing_service),
) -> NovelProcessingResponse:
    """Reprocess a failed novel."""
    novel = await novel_service.get_novel(novel_id)
    if not novel:
        raise HTTPException(status_code=404, detail="Novel not found")
    
    if not novel.file_path:
        raise HTTPException(status_code=400, detail="No file associated with this novel")
    
    file_path = Path(novel.file_path)
    if not file_path.exists():
        raise HTTPException(status_code=400, detail="File not found")
    
    # Reset status
    await novel_service.mark_status(novel_id, "pending")
    
    # Process in background
    async def process_with_progress(novel_id: str, file_path: Path):
        async def progress_callback(progress: float, message: str):
            await novel_service.update_processing_progress(novel_id, progress=progress, message=message)
        
        try:
            await text_service.process_novel(novel_id, file_path, progress_callback)
        except Exception as e:
            logger.exception(f"Failed to reprocess novel {novel_id}")
            await novel_service.update_processing_progress(
                novel_id, progress=0.0, message=f"处理失败: {str(e)}"
            )
    
    background_tasks.add_task(process_with_progress, novel_id, file_path)
    
    return NovelProcessingResponse(
        message="Novel reprocessing started",
        novel_id=novel_id,
        status="processing",
    )
```
