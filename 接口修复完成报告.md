### 3. 搜索接口方法调整

**文件**：`frontend/src/utils/api.ts`

**修改前**（GET + Query String）：
```typescript
async search(params: {
  query: string;
  novelIds?: string[];
  mode?: 'keyword' | 'semantic';  // ❌ 后端不需要
  topK?: number;
}): Promise<SearchResult> {
  const queryParams = new URLSearchParams();
  queryParams.append('query', params.query);
  // ... 构建Query String
  return apiRequest<SearchResult>(`/search?${queryParams.toString()}`);
}
```

**修改后**（POST + JSON Body）：
```typescript
async search(params: {
  query: string;
  novelIds?: string[];
  topK?: number;
  includeReferences?: boolean;  // ✅ 新增
  saveHistory?: boolean;        // ✅ 新增
}): Promise<SearchResult> {
  return apiRequest<SearchResult>('/search', {
    method: 'POST',
    body: JSON.stringify({
      query: params.query,
      novelIds: params.novelIds || [],
      topK: params.topK || 5,
      includeReferences: params.includeReferences !== false,
      saveHistory: params.saveHistory !== false,
    }),
  });
}
```

**组件调整**：`frontend/src/pages/SearchPage.tsx`

**修改前**：
```typescript
const result = await searchAPI.search({
  query,
  novelIds: selectedNovelIds,
  mode: searchMode,  // ❌ 移除
  topK: 5
});
```

**修改后**：
```typescript
const result = await searchAPI.search({
  query,
  novelIds: selectedNovelIds,
  topK: 5,
  includeReferences: true,  // ✅ 新增
  saveHistory: true         // ✅ 新增
});
```

**优势**：
- ✅ 符合POST方法语义（搜索是有副作用的操作）
- ✅ JSON Body更适合复杂参数
- ✅ 支持搜索历史保存功能

## 📝 开发者须知

### 环境变量配置

在开始开发前，请确保配置正确的环境变量：

**开发环境** (`frontend/.env`):
```env
VITE_API_BASE_URL=http://localhost:8000/api/v1
VITE_APP_TITLE=小说RAG分析系统
```

**生产环境** (`frontend/.env.production`):
```env
VITE_API_BASE_URL=https://your-api-domain.com/api/v1
VITE_APP_TITLE=小说RAG分析系统
```

### 后端CORS配置

确保后端CORS配置包含前端域名：

```python
# backend/app/main.py
app.add_middleware(
    CORSMiddleware,
    allow_origins=[
        "http://localhost:5173",      # Vite开发服务器
        "http://localhost:3000",      # React开发服务器（备用）
        "http://127.0.0.1:5173",
        "http://127.0.0.1:3000",
        # 生产环境需要添加实际的域名
        # "https://your-frontend-domain.com",
    ],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# 包含API路由（注意 prefix="/api/v1"）
app.include_router(api_router, prefix="/api/v1")
```
